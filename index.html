常用命令

  1. 系统更新
    apt-get update
    apt-get upgrade
  2. 开启服务
    service * start
  3. 安装工具包
    apt-cache search 软件包名称
    在软件仓库查找某个软件包名称
    apt-cache show
    查看软件包详细信息
    apt-get install
    安装软件包
